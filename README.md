# Portfólio Pessoal

Este projeto é uma página web desenvolvida para apresentar meu perfil, tecnologias, projetos e formas de contato.

## Tecnologias utilizadas
- HTML
- CSS
- JavaScript

## Funcionalidades
- Apresentação pessoal
- Exibição de tecnologias
- Listagem de projetos
- Seção de contato

## Objetivo
Criar uma página simples e organizada para compartilhar informações profissionais e projetos desenvolvidos.
